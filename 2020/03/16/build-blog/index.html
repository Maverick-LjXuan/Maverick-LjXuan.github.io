分享一下自己用hexo搭建nexmoe主题博客的过程
主要是写一些搭建过程中遇到的坑，因为网上都有详细的教程，自己也是看网上教程弄的，所以就不多赘述
以后还会不断地完善博客的功能，到时候也会记录下来,不定期更新

前期准备

大概需要下面这些东西，先提及一下

  1. 注册github账号
  2. Git for windows // 我的环境是windows
  3. 安装 NodeJs
  4. 安装hexo
    ……

Git bash 安装

从Git官网下载git bash
但是国内网站下载很慢，也可以用Pandownload下载百度网盘资源 //当前版本2.25
提取码 at9j，当然如果你有vip就不用pandownload了
安装包一路’next’就好

装好之后可以直接在文件夹右键’Git Bash Here’

NodeJs 安装

下载地址 选择LTS（长期稳定版）就ok
下载后之后安装包也是一路’next’，不过在 CustomSetup 这步要选择 Add to Path

关于图床

建议使用七牛云
实名注册，添加对象存储
上传图片就可以使用图片的链接而且不会挂了
